# BookCompanion

## Requirements
An Android Emulator/Phone is required to run this application. I personally used Android Studio. Node JS is also required.




## Expo Installation

```bash
#Windows
npm install --global expo-cli
#Mac
npm i -g expo-cli
#or
sudo npm i -g expo-cli
```

## Running the app

```bash
#npm
npm install 
expo start
```
