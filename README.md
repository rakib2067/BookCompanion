# BookCompanion

## Technologies

- Javascript
- React
- CSS
- Firebase 

## Requirements

- An Android Emulator/Phone is required to run this application.
- NodeJS


## Expo Installation

```bash
#Windows
npm install --global expo-cli
#Mac
npm i -g expo-cli
#or
sudo npm i -g expo-cli
```

## Running the app

```bash
#npm
npm install 
expo start
```
